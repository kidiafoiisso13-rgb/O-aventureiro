# Glitch Runner

## 📝 Breve Resumo do Projeto
Glitch Runner é um jogo de aventura e plataforma top-down 2D 
ambientado no núcleo de uma simulação de realidade virtual em 
colapso. O jogador controla o "Debugger", um avatar digital que 
precisa navegar por cenários corrompidos usando o 
Dash de Fragmentação (uma mecânica de glitch que permite 
aumentar a velocidade e atravessar temporariamente obstáculos específicos), 
coletar orbes de dados perdidos e alcançar o portal de extração para estabilizar 
o sistema antes que todo o setor seja deletado.

## 🎓 Contexto de Desenvolvimento
Este jogo é um projeto acadêmico de caráter avaliativo, 
produzido e desenvolvido inteiramente durante as aulas práticas 
de desenvolvimento de jogos. O objetivo do projeto foi aplicar 
conceitos reais de física 2D, manipulação de interface de usuário 
(UI), sistemas de inputs modernos, gerenciamento de colisões por 
gatilhos (`Triggers`) e controle de versão através do Git/GitHub.
